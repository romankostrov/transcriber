1. ВОЗМОЖНОСТИ
================================================================================

  • Поддержка форматов: WEBM, MP4, WAV, MP3, M4A, MKV
  • Пакетная обработка нескольких файлов
  • GPU ускорение (CUDA для NVIDIA)
  • Таймкоды в каждом фрагменте
  • Экспорт в TXT, SRT, JSON
  • Оптимизировано для русского языка
  • Drag & Drop файлов
  • Portable версия для USB-флешки

================================================================================
2. ТРЕБОВАНИЯ
================================================================================

  • Windows 10 или новее
  • 4GB RAM (рекомендуется 8GB+)
  • Свободное место: 2-4 GB (включая модели)
  • Интернет для первого скачивания модели

================================================================================
3. УСТАНОВКА
================================================================================

  ПРОСТОЙ СПОСОБ (рекомендуется):

    1. Распакуйте архив в любую папку
    2. Запустите bootstrap_windows.bat
    3. Дождитесь завершения (5-10 минут)

  ПОРТАТИВНЫЙ СПОСОБ (для USB-флешки):

    1. Запустите make_portable.bat
    2. Готовая папка TranscriberPro_Portable/
    3. Скопируйте её на флешку

================================================================================
4. СКАЧИВАНИЕ МОДЕЛЕЙ
================================================================================

  После установки нужно скачать хотя бы одну модель.

  БЫСТРЫЙ ВАРИАНТ (~500 MB):
    Запустите download_small.bat

  КАЧЕСТВЕННЫЙ ВАРИАНТ (~1.5 GB):
    Запустите download_medium.bat

  МАКСИМАЛЬНЫЙ ВАРИАНТ (~3 GB):

  └────────────┴──────────┴────────────┴────────────────────────────────┘

7. РЕЖИМЫ РАБОТЫ
================================================================================

  БЫСТРО / SMALL (режим по умолчанию)
    - Модель: small
    - Beam size: 1
    - Compute: int8
    - Скорость: максимальная
    - Точность: базовая

  КАЧЕСТВЕННО / MEDIUM (рекомендуется)
    - Модель: medium
    - Beam size: 5
    - Compute: int8
    - Скорость: средняя
    - Точность: высокая

  МАКСИМУМ / LARGE-V3
    - Модель: large-v3
    - Beam size: 5
    - Compute: int8
    - Скорость: низкая
    - Точность: максимальная

================================================================================
8. НАСТРОЙКИ
================================================================================

  ДЛЯ ПРОЦЕССОРА (CPU):
    Device: cpu
    Compute: int8
    Beam size: 1-5

  ДЛЯ ВИДЕОКАРТЫ NVIDIA (GPU):
    Device: cuda
    Compute: float16
    Beam size: 3-5
    (Требуется установленный CUDA Toolkit)

9. PORTABLE ВЕРСИЯ
================================================================================

  СБОРКА:
    make_portable.bat

  СТРУКТУРА:
    TranscriberPro_Portable/
    ├── TranscriberPro.exe      # Главный файл
    ├── download_small.bat      # Скачать small модель
    ├── download_medium.bat     # Скачать medium модель
    ├── download_large.bat      # Скачать large модель
    ├── bin/
    │   ├── ffmpeg.exe
    │   └── ffprobe.exe
    └── models/                  # Папка для моделей

  РАЗМЕР:
    Без моделей: ~75 MB
    С small моделью: ~575 MB
    С medium моделью: ~1.6 GB
    С large моделью: ~3.1 GB

  ИСПОЛЬЗОВАНИЕ:
    1. Скопируйте папку TranscriberPro_Portable на флешку
    2. На любом компьютере запустите download_small.bat (один раз)
    3. Затем запустите TranscriberPro.exe

10. ФОРМАТЫ ВЫВОДА

    2

    Текст второго фрагмента

      "source_file": "meeting.webm",
      "model": "small",
      "segments": [

11. УСТРАНЕНИЕ ПРОБЛЕМ
================================================================================

  ПРОБЛЕМА: "Модель не найдена"
    РЕШЕНИЕ: Запустите download_small.bat

  ПРОБЛЕМА: "FFmpeg не найден"
    РЕШЕНИЕ: Запустите bootstrap_windows.bat

  ПРОБЛЕМА: Программа зависла
    РЕШЕНИЕ: Запустите cleanup.bat

  ПРОБЛЕМА: CUDA не работает
    РЕШЕНИЕ: Используйте device=cpu

  ПРОБЛЕМА: Ошибка при переносе папки
    РЕШЕНИЕ: Используйте Portable версию (make_portable.bat)

  ПРОБЛЕМА: Не хватает места на диске
    РЕШЕНИЕ: Используйте только small модель (~500 MB)
