## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🎬 **Форматы** | WEBM, MP4, WAV, MP3, M4A, MKV |
| 📦 **Пакетная обработка** | Несколько файлов за раз |
| ⚡ **GPU ускорение** | CUDA support для NVIDIA |
| 📝 **Таймкоды** | Точное время каждого фрагмента |
| 💾 **Экспорт** | TXT, SRT, JSON |
| 🌐 **Русский язык** | Оптимизировано для русского |
| 🖱️ **Drag & Drop** | Просто перетащите файлы |
| 📁 **Portable** | Работает с флешки |

🎮 Режимы работы
Быстро / small
yaml
Модель: small
Beam size: 1
Compute: int8
Скорость: ⚡⚡⚡
Качество: ⭐⭐
Для кого: Черновики, быстрая проверка, длинные записи

Качественно / medium
yaml
Модель: medium
Beam size: 5
Compute: int8
Скорость: ⚡⚡
Качество: ⭐⭐⭐⭐
Для кого: Деловые встречи, интервью, лекции

Максимум / large-v3
yaml
Модель: large-v3
Beam size: 5
Compute: int8
Скорость: ⚡
Качество: ⭐⭐⭐⭐⭐
Для кого: Судебные записи, медицинские диктовки, где важна точность

⚙️ Настройки
CPU (рекомендуется для большинства)
text
Device: cpu
Compute: int8
Beam size: 1-5 (чем больше, тем точнее, но медленнее)
NVIDIA GPU
text
Device: cuda
Compute: float16
Beam size: 3-5
Требуется установленный CUDA Toolkit и драйверы NVIDIA

🐛 Устранение проблем
❌ Модель не найдена
batch
download_small.bat
❌ FFmpeg не найден
Запустите bootstrap_windows.bat или положите ffmpeg.exe в папку bin/

❌ CUDA не работает
Установите CUDA Toolkit

Используйте device=cpu

❌ Программа зависла
batch
cleanup.bat
